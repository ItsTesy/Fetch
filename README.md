# Fetch

Python script for displaying basic system info on Linux.

## What it does

Runs a simple terminal “fetch” style overview of:

- Username & hostname  
- Distribution name  
- System uptime  
- Kernel version  
- Host model  
- RAM usage  
- Terminal type  
- Screen resolution  
- CPU model  
- GPU info  

## Requirements

- Python  
- pyautogui  
- Linux environment  

## Usage

```bash
python fetch.py
